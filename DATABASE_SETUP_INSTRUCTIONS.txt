2. TABLE CREATION:
   Create the following tables in the Masroo3k database:

   TABLE 1: Users
   Fields:
   - Id (INT, AUTO_INCREMENT, PRIMARY KEY)
   - Name (VARCHAR(255), NOT NULL)
   - Email (VARCHAR(255), NOT NULL, UNIQUE)
   - PasswordHash (VARCHAR(255), NOT NULL)
   - Role (VARCHAR(50), NOT NULL, DEFAULT 'user')
   - CreatedAt (DATETIME, NOT NULL, DEFAULT CURRENT_TIMESTAMP)
   
   Index:
   - IX_Users_Email on Email column

   TABLE 2: Templates
   Fields:
   - Id (INT, AUTO_INCREMENT, PRIMARY KEY)
   - Name (VARCHAR(255), NOT NULL)
   - Description (TEXT)
   - Category (VARCHAR(100), NOT NULL, DEFAULT 'General')
   - Duration (INT, NOT NULL, DEFAULT 30)
   - IsPopular (BOOLEAN, NOT NULL, DEFAULT FALSE)
   - CreatedAt (DATETIME, NOT NULL, DEFAULT CURRENT_TIMESTAMP)

   TABLE 3: Analyses
   Fields:
   - Id (INT, AUTO_INCREMENT, PRIMARY KEY)
   - Title (VARCHAR(255), NOT NULL)
   - Content (TEXT)
   - Score (INT, NOT NULL, DEFAULT 0)
   - RiskLevel (VARCHAR(50), NOT NULL, DEFAULT 'Medium')
   - SuccessPercent (INT, NOT NULL, DEFAULT 0)
   - Investment (DECIMAL(18,2), NOT NULL, DEFAULT 0)
   - ExpectedROI (DECIMAL(18,2), NOT NULL, DEFAULT 0)
   - CreatedAt (DATETIME, NOT NULL, DEFAULT CURRENT_TIMESTAMP)
   - ExecutiveSummary (TEXT)
   - KeyFindings (TEXT)
   - Recommendations (TEXT)
   - OwnerId (INT, NOT NULL)
   - TemplateId (INT)
   
   Foreign Keys:
   - OwnerId references Users(Id) with CASCADE DELETE
   - TemplateId references Templates(Id) with SET NULL

   TABLE 4: ActivityLogs
   Fields:
   - Id (INT, AUTO_INCREMENT, PRIMARY KEY)
   - Action (VARCHAR(100), NOT NULL)
   - EntityType (VARCHAR(100), NOT NULL)
   - EntityId (INT)
   - Description (TEXT, NOT NULL)
   - Details (TEXT)
   - IpAddress (VARCHAR(45), NOT NULL)
   - UserAgent (TEXT, NOT NULL)
   - CreatedAt (DATETIME, NOT NULL, DEFAULT CURRENT_TIMESTAMP)
   - Severity (VARCHAR(20), NOT NULL, DEFAULT 'Info')
   - UserId (INT)
   
   Foreign Keys:
   - UserId references Users(Id) with SET NULL
   
   Indexes:
   - IX_ActivityLogs_CreatedAt on CreatedAt column
   - IX_ActivityLogs_Action on Action column
   - IX_ActivityLogs_Severity on Severity column

   TABLE 5: Notifications
   Fields:
   - Id (INT, AUTO_INCREMENT, PRIMARY KEY)
   - UserId (INT, NOT NULL)
   - Title (VARCHAR(255), NOT NULL)
   - Message (TEXT, NOT NULL)
   - Type (VARCHAR(20), NOT NULL, DEFAULT 'info')
   - IsRead (BOOLEAN, NOT NULL, DEFAULT FALSE)
   - CreatedAt (DATETIME, NOT NULL, DEFAULT CURRENT_TIMESTAMP)
   - ActionUrl (VARCHAR(500))
   
   Foreign Keys:
   - UserId references Users(Id) with CASCADE DELETE
   
   Indexes:
   - IX_Notifications_CreatedAt on CreatedAt column
   - IX_Notifications_IsRead on IsRead column
